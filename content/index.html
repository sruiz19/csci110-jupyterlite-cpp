Getting Started

Welcome to our interactive coding environment! This website allows you to practice C++ directly in your browser without installing software.

To begin:

  1. Open the File Browser on the left side.
  2. Select a notebook from the module folders.
  3. Run code cells using Shift + Enter.

Important Tips

  • If you see a redefinition of main error: go to Kernel → Restart Kernel and Clear Outputs.
  • Your work is saved in your browser only. Download important files regularly.
  • Larger assignments should still be completed in OnlineGDB.
